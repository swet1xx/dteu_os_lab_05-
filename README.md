# Лабораторна робота №5. Робота з системою контролю версій Git

**Тема:** Розгалужена робота у системі контролю версій Git. Робота з віддаленими репозиторіями.

**Опис:** Ця лабораторна робота присвячена освоєнню продвинутих механіків Git...

**Мета:** Опанувати практичні навички розгалуженої роботи в Git...

## Контрольні питання

1. **Що таке гілка в Git?**
   Гілка в Git - це легкий переміщуваний вказівник на один з комітів. Вона дозволяє розробникам відгалужуватися від основної лінії розробки.

2. **Яка різниця між 'git merge' та 'git rebase'?**
   'git merge' створює новий коміт-злиття, який об'єднує історії двох гілок. 'git rebase' переміщує послідовність комітів однієї гілки в основу іншої.
